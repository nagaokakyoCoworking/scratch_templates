## Inkscape 用ドキュメントテンプレート

Sctach のキャラクタを作成する際に、Scratch 付属の描写ツールでは、ちょっと物足りないって思った人向け

Inkscape/scratch_inkscape_template.svg

### インストール方法

C:\Program Files\Inkscape\share\templates\ にダウンロードした scratch_inkscape_template.svg をコピーをする

### 使い方

メニュー「ファイル」→「New from template...」を選択すると、”テンプレートから作成”ダイアログが表示されます。
そのリストから "scratch inkscape template" を選択すると使うことができます。
